## How to run the app

### 1. Clone the repository

git clone https://github.com/Florence100/simple-books-catalogue.git

cd simple-books-catalogue

### 2. Install dependencies

npm install

### 3. Run development server

npm run start

The application will start on:

http://localhost:3000

### 4. Build production version

npm run build

The production files will be generated in the `dist` folder.

## Project Structure

src  
├── components — reusable UI components (e.g., book card creation)  
├── dto — data transfer objects used to normalize API data before using it in the UI  
├── handlers — event handlers for user interactions (search, favorites, etc.)  
├── services — modules responsible for external data and storage  
├── state — application state logic  
├── ui — rendering logic for updating the DOM   
├── styles — CSS files  
├── assets — static files  
├── index.js — application entry point  
└── index.html — main HTML template

dist  
Production build generated by Webpack.
